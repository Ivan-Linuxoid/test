# 🚀 О проекте

**ROSik BIBA** — многофункциональная мобильная робототехническая платформа, созданная для решения задач дистанционного управления, взаимодействия с объектами и телеприсутствия.

Проект объединяет:

- механику
- силовую электронику
- embedded-программирование
- системы удалённого управления
- роботизированные манипуляторы
- real-time видеостриминг

Основная цель проекта — создание мощного и устойчивого мобильного робота, способного выполнять широкий спектр инженерных и прикладных задач.

---

# ✅ Основные возможности

ROSik BIBA способен:

- перемещаться по рабочей площадке
- выполнять точное управление скоростью
- поднимать рабочую платформу по вертикали
- управлять двумя манипуляторами
- захватывать и удерживать объекты
- передавать live-видео с камеры
- воспроизводить звук через встроенную аудиосистему
- выполнять удалённые операции через пульт управления

---

# 🏗 Архитектура робота

Система состоит из нескольких основных подсистем:

```text
Пульт управления
        ↓
Канал передачи данных
        ↓
Raspberry Pi (центральный сервер)
        ↓
Контроллеры и драйверы
        ↓
Исполнительные механизмы
````

Подсистемы:

* мобильная платформа
* силовая электроника
* подъёмный механизм
* манипуляторы
* система связи
* видеосистема
* аудиосистема

# ⚙ Механическая конструкция

## Мобильная платформа

<img src="assets/robot_side.jpg" width="800"/>

Основой робота является усиленная четырёхколёсная платформа.

Основные требования при проектировании:

* высокая жёсткость конструкции
* устойчивость
* правильное распределение массы
* удобство обслуживания

---

## Колёсная база

В качестве приводных колёс используются колёса от гироскутеров.

Преимущества:

* высокий крутящий момент
* хорошая грузоподъёмность
* плавный старт
* высокая надёжность

Четырёхколёсная схема обеспечивает устойчивость даже при поднятом подъёмнике.

---

## Подъёмный механизм

<img src="assets/lift.jpg" width="700"/>

Подъёмник реализован на базе шагового двигателя.

Основные задачи:

* вертикальное перемещение рабочей зоны
* увеличение рабочей области манипуляторов
* точное позиционирование

Шаговый двигатель обеспечивает:

* высокую точность
* повторяемость движений
* плавность работы

---

## Манипуляторы

Робот оснащён двумя манипуляторами.

Манипуляторы позволяют:

* захватывать предметы
* переносить объекты
* взаимодействовать с окружающей средой

Подъёмник и манипуляторы работают как единая система.

Включает:

* драйверы двигателей
* распределение питания

* защиту от перегрузок

## Исполнительные механизмы

К исполнительным узлам относятся:

* мотор-колёса

* аудиомодуль
* камера

---

# 🎮 Система управления

ROSik BIBA поддерживает удалённое управление через кастомный пульт.

Оператор может управлять:

* движением платформы
* скоростью
* направлением
* подъёмником
* манипуляторами
* дополнительными функциями

Система проектировалась с акцентом на минимальную задержку управления.

---

# 📡 Передача данных

Обмен данными между пультом и роботом реализован через собственный протокол передачи данных.

Протокол обеспечивает:

* устойчивый обмен пакетами
* передачу управляющих команд
* синхронизацию состояний
* обработку ошибок передачи

Поддерживается real-time взаимодействие между оператором и роботом.

---

# 🎥 Видеостриминг

Робот оснащён камерой с передачей изображения в реальном времени.

Видеостриминг позволяет:

* наблюдать окружение
* точно управлять роботом
* выполнять дистанционные операции

Live-поток минимизирует задержки при работе.

---

# 🔊 Аудиосистема

В робота встроена полноценная аудиосистема.

Состав:

* магнитола
* усилитель
* динамики

Назначение:

* воспроизведение звука
* голосовая связь
* сигнальные уведомления
